rank_extractor — Processing of PCIM data¶

This module offers a way to process the output of ParserBoinc.py and aggregate all the rankings in a list ready to be postrocessed. The functions of this module use ony the .expansion files

extractor(input_folder, leng)¶

take in input a string input_folder containing a path of a directory containing the output of ParserBoinc of many PCIM, an integer leng that is the len of the ranks to extract. Return a list of ranks taken from every PCIM in the input folder.
